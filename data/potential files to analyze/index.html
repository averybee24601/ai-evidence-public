``` ## Features - **Single HTML file**: Self-contained with Three.js via CDN. - **File Input**: Select any audio file (MP3, WAV, etc.). - **3D Sphere Mesh**: High-detail `SphereGeometry(2, 64, 32)` with ~6k vertices. - **Audio Visualization**: - Radial displacement of vertices based on frequency data (low freq bottom, high top). - Vertex colors shift hue (blue-low to red-high) with amplitude. - **Controls**: OrbitControls for mouse drag/zoom/pan. - **Auto-rotate**: Slow sphere rotation. - **Play/Loop**: Loads, plays, loops audio; toggle stop. - **Responsive**: Fullscreen, resizes on window change. - **UI**: Overlay with status. - **Performance**: Dynamic position/color updates, vertex normals recomputed each frame (smooth on modern hardware).
